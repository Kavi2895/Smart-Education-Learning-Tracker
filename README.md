# 🎓 Smart Education Learning Tracker

## 📌 Project Overview

**Smart Education Learning Tracker** is a web-based application designed to help students track their study progress, monitor performance, and receive learning guidance. The system allows users to record study hours, subjects, and marks, visualize performance through charts, and generate study reports.

This project demonstrates a **full-stack application** integrating frontend visualization, backend APIs, and database storage.

---

## 🚀 Key Features

### 📊 Study Progress Tracking

Students can record:

* Student Name
* Subject
* Study Hours
* Marks

All study records are stored and displayed dynamically.

### 📈 Performance Visualization

The system uses charts to display student marks, helping users easily analyze their learning progress.

### 📑 PDF Report Generation

Users can download a **study report in PDF format**, containing all student study records.

### 🧠 Learning Guidance

Based on the marks entered, the system provides **automatic learning suggestions** to help students improve their performance.

### 🔍 Search Function

Students can quickly search and filter study records.

## 🏗️ System Architecture

Frontend → Backend API → Database

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js + Express.js
* **Database:** MySQL
* **Charts:** Chart.js
* **PDF Generation:** jsPDF

---

## 🔄 Data Flow of the System

The application follows a **client–server architecture**.

1. User enters study details in the frontend form.
2. The frontend sends the data to the backend API.
3. The backend server processes the request.
4. Data is stored in the MySQL database.
5. The frontend retrieves and displays the data.
6. Charts and guidance are generated automatically.
